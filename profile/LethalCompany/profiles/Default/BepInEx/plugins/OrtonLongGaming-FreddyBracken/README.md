# Freddy Bracken
Replaces the Bracken (Flowerman) with Freddy Fazbear.

# Manual Install Instructions
- Install BepInEx
 
- download manual download zip 

- drag freddy.bundle and FreddyBracken.dll into the Plugins folder in BepInEx
